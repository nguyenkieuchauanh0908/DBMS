Database được lưu trong folder Database dùng chung cho cả bài 1 và 2
Để tạo lại CSDL có 2 cách:
C1: Dùng file .sql trong folder database tạo lại CSDL
C2: Dùng file .bak trong folder database để restore CSDL
Tham khảo cách làm: https://learn.microsoft.com/vi-vn/sql/relational-databases/backup-restore/quickstart-backup-restore-database?view=sql-server-2017 
Chú ý: khi chạy chương trình phải thay đổi chuỗi kết nối (Connection string ở form1)
